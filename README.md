# 闲鱼数据采集助手

Chrome浏览器插件，快速采集闲鱼商品数据，精细化运营店铺。
![插件截图](docs/index.png)

| 功能模块 | 当前状态 | 说明 |
|---------|---------|------|
| 关键词采集商品列表数据 | ✅ 已实现 | 输入关键词，自动爬取闲鱼商品数据 |
| CSV数据导出 | ✅ 已实现 | 一键导出采集数据为CSV文件 |
| 商品详情采集 | ✅ 已实现 | 采集商品的浏览量、想要数据、卖家数据采集 |
| 商品一键填充 | ✅ 已实现 | 自动填充图片、描述、价格，快速发布商品 |
| 飞书表格集成 | ✅ 已实现 | 支持一键上传到飞书表格，实现精细化运营 |
| 集成AI | ⏳ 计划中 | 实现一键改写文案、根据聊天上下文内容回复消息 |

## 🚀 快速开始ß

### 安装插件

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹即可

### 使用步骤

#### 关键词采集

1. 打开 https://www.goofish.com/ 并登录
2. 点击浏览器右上角插件图标
3. 输入搜索关键词（如：雨伞）
4. 设置采集页数（建议1-10页）
5. 点击“开始爬取”
6. 采集完成后点击“导出CSV”

#### 商品详情采集 + 一键发布 ✨

1. **采集商品**
   - 打开任意商品详情页（`https://www.goofish.com/item?id=xxx`）
   - 点击右下角黄色“⬇️”下载按钮
   - 商品数据自动保存到本地

2. **查看采集记录**
   - 点击右下角黑色“📋”控制台按钮
   - 查看所有已下载的商品记录
   - 支持刷新、清空、删除操作

3. **一键填充** 🚀
   - 在控制台表格中点击“🚀 发布”按钮
   - 自动打开发布页面并填充：
     - ✅ 主图（第1张图片）
     - ✅ 详情图（第2张及之后的图片）
     - ✅ 商品描述
     - ✅ 价格和原价（原价 = 价格 × 5）
   - 检查信息后点击发布即可

### 商品详情采集
- 在商品详情页一键下载完整数据
- 自动保存到本地存储
- 支持批量采集多个商品

### 智能控制台
- 美观的表格展示（黄黑主题）

- 支持刷新、清空、删除操作
- 显示商品图片、描述、浏览量、想要人数等

### 一键自动发布 🚀
- 智能识别商品信息
- 自动上传主图和详情图
- 自动填充商品描述
- 自动计算和填充价格
- 实时日志显示，进度可视化

### 采集数据字段

商品详情采集包含：
- 商品ID、标题、价格
- 卖家昵称、唯一名
- 商品描述
- 商品图片（多张）
- 想要人数、浏览量、收藏数
- 发布时间、下载时间
- 商品分类、地区
- 完整的原始数据

- LocalStorage 数据存储
- Fetch API 图片处理
